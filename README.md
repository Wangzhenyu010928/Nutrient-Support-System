# -核心算法：DEA-CCR 模型
本系统采用 DEA-CCR 模型（由 Charnes, Cooper 和 Rhodes 提出），通过线性规划求解多个决策单元（DMUs，即不同食物）的相对效率。

投入指标 (Inputs)：单位成本、热量、脂肪含量等。

产出指标 (Outputs)：蛋白质、维生素、矿物质等营养成分。

目标函数：在给定投入下最大化产出效率，识别出处于“效率前沿面 (Efficiency Frontier)”的帕累托最优食物组合。

VBA 实现逻辑：
由于 Excel 默认不直接支持批量 DEA 运算，我通过 VBA 编写了循环求解脚本，自动遍历所有食物条目并调用 Solver 引擎，将复杂的数学规划过程转化为用户的一键式操作。
